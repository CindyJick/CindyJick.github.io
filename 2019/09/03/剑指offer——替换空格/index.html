剑指offer——替换空格
2019-09-03|技术

一. 题目

题目链接

请实现一个函数，将一个字符串中的每个空格替换成“%20”。例如，当字符串为We Are Happy.则经过替换之后的字符串为We%20Are%20Happy。

二. 思路

1. 初始思路

遍历一遍字符串的字符，如果不是空格则将该字符新增到新的str中，如果是空格则将”%20”增加到新的str中——无耻的用了StringBuffer，好像违背了题目的本意

2. 参考思路

1). 先循环一次计算出字符中的空格数
2). 然后从字符串的尾端开始遍历字符
3). 记录当前遇到的空格数
4). 对于非空格字符 => 新位置 = 旧位置 + (3 - 1)*空格数
5). 空格字符 => 用”%20”覆盖
