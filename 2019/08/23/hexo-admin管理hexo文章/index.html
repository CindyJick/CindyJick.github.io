hexo-admin管理hexo文章
2019-08-23

1.安装hexo-admin

键入npm install –save hexo-admin

2.重新启动hexo

键入hexo server -d

3.访问admin页面

在浏览器上访问 yourip:4000/admin

4.设置amdin登陆拦截

  1. 先在admin页面中点击”Settings”选项卡
  2. 接着点击”Setup authentification here”
  3. 填入设置”用户名”、”密码”、”密文”
